# Class Feature Analysis — Levels 1–5

Analysis of all class and subclass features for levels 1–5 across all 13 PHB/official classes. Each feature is categorised for combat automation purposes.

**Categories:**
- **COMBAT_ACTIVE** — Needs a button in encounter UI (action/bonus action/reaction usage)
- **COMBAT_PASSIVE** — Always-on modifier, tracked but no button needed
- **COMBAT_MODIFIER** — Modifies how another action works
- **RESOURCE** — Has limited uses needing tracking (noted alongside another category when applicable)
- **FLAVOUR** — No combat relevance
- **SPELLCASTING** — Grants spell slots and spell list access
